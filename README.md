# gemma.cpp nix

[`gemma.cpp`](https://github.com/google/gemma.cpp) packaged with [nix](https://zero-to-nix.com/concepts/nix/).

## Usage

```shell
$ nix run github:omarjatoi/gemma-cpp-nix
```
